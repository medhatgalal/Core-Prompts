# Steering Files

This directory holds durable Kiro steering for repo workflow and documentation governance.

Use these files as the canonical policy layer for cross-surface operating rules that should guide both Kiro and Codex work in this repository.

- `repo-workflow.md`: branch hygiene, mutation sequencing, and safe repo-change workflow
- `docs-governance.md`: documentation hierarchy, docs drift rules, and generated-doc positioning
- `agent-behavior.md`: rule-writing discipline, non-commingling of docs and rules, and ask-evaluation quality
