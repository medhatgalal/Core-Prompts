# Assets Index

Use this directory for documentation visuals that have instructional value.

## Recommended contents

- Hero image assets used by README or docs landing pages
- Screenshot captures for onboarding or workflow explanation
- Exported diagram images (if Mermaid source is also committed)

## Requirements

- Include meaningful alt text when embedding in Markdown.
- Keep visuals factual and aligned with real repository behavior.
- Avoid purely decorative images.
